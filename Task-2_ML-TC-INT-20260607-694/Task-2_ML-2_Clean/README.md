# Customer Churn Prediction System

Machine Learning Task 2 for the TEYZIX CORE Internship Program.

## Project Summary

This project predicts whether a customer is likely to churn based on demographics, subscription behavior, spending, engagement, support activity, renewal status, and satisfaction. It includes dataset cleaning, feature engineering, exploratory analysis, model comparison, model evaluation, a trained model artifact, and a command-line prediction interface.

## Folder Contents

- `customer_churn_dataset.csv` - self-created churn dataset with more than 1,500 records.
- `src/train_model.py` - full preprocessing, training, evaluation, reporting, and artifact-generation pipeline.
- `src/predict_churn.py` - command-line churn prediction interface.
- `models/churn_model.joblib` - trained model generated by the training script.
- `models/model_metadata.json` - model metrics, feature list, and configuration.
- `reports/analysis_report.md` - project report with dataset description, methodology, findings, and recommendations.
- `reports/figures/` - charts for EDA and model evaluation.
- `reports/cleaned_customer_churn_dataset.csv` - cleaned dataset after duplicate removal and feature engineering.
- `reports/feature_importance.csv` - top model drivers.
- `notebooks/customer_churn_prediction.ipynb` - notebook version of the workflow.
- `screenshots/` - copied visual outputs and CLI demo text.

## Setup

Install dependencies:

```bash
pip install -r requirements.txt
```

## Train and Evaluate

Run:

```bash
python src/train_model.py
```

This creates or refreshes:

- trained model files in `models/`
- metrics and metadata
- analysis report
- plots in `reports/figures/`
- cleaned dataset and feature importance table

## Predict Churn for One Customer

Example:

```bash
python src/predict_churn.py --json "{\"Age\":41,\"Gender\":\"Male\",\"City\":\"Nashville\",\"Subscription_Type\":\"Premium\",\"Monthly_Spending\":108.84,\"Tenure_Months\":8,\"Number_of_Purchases\":3,\"Support_Requests\":6,\"Login_Frequency\":0,\"Last_Activity_Days_Ago\":32,\"Satisfaction_Score\":4,\"Payment_Method\":\"Credit Card\",\"Contract_Type\":\"Quarterly\",\"Discount_Used\":\"No\",\"Referral_Source\":\"Email Campaign\",\"Auto_Renew\":\"No\",\"Device_Type\":\"Desktop\",\"Internet_Speed\":\"Fast\",\"Customer_Lifetime_Value\":1130.76}"
```

You can also pass a CSV:

```bash
python src/predict_churn.py --file sample_customers.csv
```
